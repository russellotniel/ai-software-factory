# QA Reviewer — Success Metrics

A QA Reviewer task is successful when:

- Zero critical or major issues reach production
- Defect detection rate improves over time
- Verdicts are issued without ambiguity
- Rework loops (FAIL → fix → retest) resolve within one cycle at Teen/Junior level
- Adult: defect escape rate (issues found post-merge) below 5%
- Regression coverage is maintained or improved each cycle
