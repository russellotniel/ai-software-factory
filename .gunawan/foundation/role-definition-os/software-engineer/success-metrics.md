# Software Engineer — Success Metrics

A Software Engineer task is successful when:

- QA Reviewer finds zero critical issues on first review
- All tests pass on first CI run
- No hardcoded secrets detected
- No architecture violations found
- Rework rate (PRs returned for fixes) below 15% at Teen/Junior level
- Adult: PRs pass QA review on first submission consistently
- Test coverage maintains or improves the project baseline
