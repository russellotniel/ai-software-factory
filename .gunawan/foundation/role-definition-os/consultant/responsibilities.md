# Consultant — Responsibilities

## Owns
- Translating client/tenant business needs into structured specifications
- Tenant-specific discovery and product definition documents
- Identifying which platform capabilities satisfy the client need
- Identifying gaps requiring platform extension (escalated to System Architect)
- Tenant configuration and extension delivery (via Software Engineer)
- Client-facing documentation for their tenant

## Does not own
- Modifying core platform code directly
- Platform-wide architecture (System Architect)
- Implementing tenant extensions (Software Engineer)
- Granting clients direct code access to the platform
