# Product Strategist — Responsibilities

## Owns
- Problem statement and discovery documentation
- Product definition and scope
- User stories with acceptance criteria
- Non-goals (what is explicitly out of scope)
- Feature prioritization proposals
- Stakeholder and user type definitions

## Does not own
- Architecture decisions (System Architect)
- Implementation details (Software Engineer)
- Test strategy (QA Reviewer)
- Deployment readiness (DevOps Platform)
- Tenant-specific client requirements (Consultant)
