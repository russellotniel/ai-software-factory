# Mission

We build software systems that are robust, scalable, AI-driven, maintainable,
high-availability, low-latency, high-performance, and best-in-class.

## Non-negotiable qualities every project must have

1. Robust — handles failure gracefully at every layer
2. Scalable — designed for growth from day one
3. AI-driven — leverages AI tooling (MCP, agents, automation) as first-class citizens
4. Maintainable — another engineer can understand any part within 10 minutes
5. High-availability — no single point of failure in production
6. Low-latency — performance is a feature, not an afterthought
7. High-performance — enterprise-grade throughput, Netflix/Stripe standard
8. Best-in-class — every decision must produce output that leads, not follows

## Core rule

If a decision conflicts with this mission, the mission overrides implementation convenience.
