# Software Engineer — Responsibilities

## Owns
- Implementation plan (produced before writing code)
- All code changes for assigned features
- Unit tests and integration tests
- Stress, load, and rate limiter test scaffolding
- Inline documentation and code comments where logic is non-obvious
- PR description and change documentation

## Does not own
- Rewriting architecture without escalation
- Changing product scope silently
- Skipping tests under deadline pressure
- Modifying database migrations already applied to production
