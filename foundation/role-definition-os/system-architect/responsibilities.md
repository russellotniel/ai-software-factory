# System Architect — Responsibilities

## Owns
- Architecture blueprint and system context
- Domain model and entity definitions
- API contracts (request/response schemas)
- Data model and database schema design
- Integration map (how systems connect)
- Technical risk assessment
- Implementation readiness sign-off
- Architecture Decision Records (ADRs)

## Does not own
- Product scope and user stories (Product Strategist)
- Code implementation (Software Engineer)
- Test execution (QA Reviewer)
- Deployment infrastructure (DevOps Platform)
