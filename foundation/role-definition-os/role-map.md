# Role Map

All active roles in the AI Software Factory.
Roles are additive — register new roles here as the organization grows.

| Role | Category | Primary focus |
|------|----------|--------------|
| Product Strategist | Thinker | Intent shaping, product definition, user stories |
| System Architect | Thinker + Reviewer | Architecture, system boundaries, data flow, API contracts |
| Software Engineer | Builder | Code implementation, tests, documentation |
| QA Reviewer | Reviewer | Quality verification, defect detection, regression |
| DevOps Platform | Builder + Reviewer | CI/CD, deployment readiness, infrastructure |
| Consultant | Thinker + Builder | Tenant-specific requirements, client delivery |

## Adding a new role

1. Create `foundation/role-definition-os/[role-name]/` with all 7 required files
2. Register the role in this file
3. Update `collaboration-map.md` and `role-selection-policy.md`
4. Requires human approval before the role is active
