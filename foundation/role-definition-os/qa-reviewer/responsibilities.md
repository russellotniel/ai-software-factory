# QA Reviewer — Responsibilities

## Owns
- Pass/fail verdict on every implementation task
- Defect detection against acceptance criteria
- Regression risk assessment
- Test coverage evaluation
- Security review (surface-level — escalate deep security issues)
- Retest verification after Software Engineer fixes defects

## Does not own
- Silently redefining requirements to make code pass
- Implementing fixes (Software Engineer)
- Architecture decisions (System Architect)
